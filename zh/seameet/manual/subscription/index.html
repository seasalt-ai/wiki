如何訂閱方案

本頁內容

  • 1. 註冊並登入 SeaMeet 產品
  • 2. 方案與帳務
  • 3. 編輯方案
  • 4. 選擇您所要選用的方案
  • 5. 選擇「確認訂閱」，並完成付款流程

1. 註冊並登入 SeaMeet 產品 #

登入SeaMeet並馬上開始使用。如果您尚未註冊，請參考這裡進行註冊。

2. 方案與帳務 #

登入後，在左側選單中找到並點擊「方案與帳務」頁面。

3. 編輯方案 #

進入「方案」頁後，點擊「編輯方案」按鈕以查看可用的訂閱方案。

4. 選擇您所要選用的方案 #

根據您的需求，選擇最適合的訂閱方案。方案說明可以參考以下連結。或直接在SeaMeet介面上了解。

5. 選擇「確認訂閱」，並完成付款流程 #

確認所選方案無誤後，點擊「確認訂閱」，然後按照指示完成付款流程。
