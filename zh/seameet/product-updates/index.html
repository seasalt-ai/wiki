3/4/2025 #

新功能與改進 #
  • 新增以下會議記錄語言：西班牙語、法語、德語、丹麥語、波蘭語、泰語、印地語、越南語、日語、韓語
  • 調整日光時間導致的時間戳記錯誤問題
  • 修正中文維基百科的連結
  • 修正「替換說話人」導致的說話人排版位移問題
  • 調整摘要範本清單的排序問題

2/20/2025 #

新功能與改進 #
  • 新增「摘要模板」功能，用戶可為各會議套用不同的摘要模板
  • 登陸頁調整「立即嘗試」區塊的邀請按鈕文字排版
  • 調整會議清單上「一小時內」的時間戳記顯示方式
  • 修正英文用戶若註冊後數天未使用，應收到英文而非中文提示信件

  • 調整會議清單的時間戳記，將一小時之內結束或即將開始的會議，以分鐘數顯示於清單上
  • 修正擴充程式登入未註冊過的Google帳號時，會卡在載入畫面無法繼續的問題
  • 修正「說話人辨識」排序的計算方式
